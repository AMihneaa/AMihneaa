## Currently Working On

I’m currently working on my **bachelor’s thesis**, focused on a **multi-service application for signal processing and interpretation**.

The project is built around a modern architecture based on **React, FastAPI, gRPC, and separate AI services**.  
It supports **signal file upload, preprocessing, and evaluation with pre-trained models**, while the results are orchestrated through the backend and designed for future integration with a robotic platform, where predictions can be mapped into concrete behaviors.

The thesis emphasizes:

- **clean architecture**
- **separation of responsibilities**
- **scalability**
- **integration of intelligent software components into a complete end-to-end system**
